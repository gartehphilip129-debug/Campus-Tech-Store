## Project Title
Campus Tech Store – Online Electronics E-Commerce Website

---

## Project Overview
Campus Tech Store is a modern no-code e-commerce website designed using Wix. The platform provides affordable and high-quality technology products for university students and professionals. The website demonstrates key e-commerce features such as product listings, shopping cart functionality, and a responsive user interface.

---

## Platform Used
- Wix (No-Code Website Builder)

---

## Features Implemented
- Responsive website design (mobile, tablet, desktop)
- Product catalog with images, prices, and descriptions
- Shopping cart functionality (add to cart and view items)
- Contact form for customer inquiries
- About page with mission, vision, and values
- Navigation menu (Home, Products, About, Contact, Cart)
- Featured products section
- Newsletter subscription section
